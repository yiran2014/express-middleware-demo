#### 创建express空环境（只有node_modules和package.json）
```
$ mkdir express_middleware_demo
$ cd express_middleware_demo
$ npm init  //使用 npm init 命令为应用程序创建 package.json 文件
$ npm install express --save  
//采用 --save 选项安装的 Node 模块已添加到 package.json 文件中的 dependencies 列表。 今后运行目录中的 npm install 将自动安装依赖项列表中的模块
```
